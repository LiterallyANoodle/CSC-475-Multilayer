# CSC-475-Multilayer

This is the repository for the AI multilayer network project

## To build from source:

# On windows...
Open a command line in the top folder of the project and run

```
.\gradlew build
```

Then run

```
java -jar .\build\libs\CSC-475-Multilayer.jar
```

# On linux...
Open a command line in the top folder of the project and run

```
./gradlew build
```

Then run

```
java -jar ./build/libs/CSC-475-Multilayer.jar
```
